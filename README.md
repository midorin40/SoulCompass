# SoulCompass

SoulCompass は、対話・性格傾向・動機づけ指標を組み合わせて、自分の状態を静かに見つめ直すためのセルフリフレクション Web アプリです。

![SoulCompass screenshot](./public/%E3%82%B9%E3%82%AF%E3%83%AA%E3%83%BC%E3%83%B3%E3%82%B7%E3%83%A7%E3%83%83%E3%83%88%202026-03-23%20132505.png)

## Features

- チャット形式で内省をサポート
- Big Five の可視化
- Motivation Axes の可視化
- 落ち着いたダーク UI による集中しやすい体験

## Local Development

### Prerequisites

- Node.js

### Setup

1. `npm install`
2. `.env.local` に `GEMINI_API_KEY` を設定
3. `npm run dev`

## Build

`npm run build`
